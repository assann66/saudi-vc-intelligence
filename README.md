# Saudi VC Intelligence Platform

Executive intelligence dashboard for the Saudi Arabian venture capital ecosystem.

**Live:** [saudi-vc-intelligence.vercel.app](https://saudi-vc-intelligence.vercel.app)

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS with glassmorphism dark theme
- **Charts:** Recharts (bar, radar, area charts)
- **Icons:** Lucide React
- **Language:** TypeScript

## Features

- Executive dashboard with 7 KPI cards (Sector Attractiveness, Company Investability, Risk, Market Gap, Funding Momentum, Competition Intensity, Saudi Relevance)
- Interactive sector analysis with bar and radar chart views
- Cumulative funding trends area chart
- Sortable company ranking table
- Sector deep-dive page with performance metrics
- Company directory with search, filtering, and sorting
- Premium dark theme with Bloomberg Terminal-inspired design

## Data

- 10 Saudi VC sectors with comprehensive metrics
- 28 companies with funding, stage, growth, and investor data

## Getting Started

```bash
npm install --include=dev
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
